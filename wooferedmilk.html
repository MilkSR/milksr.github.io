General

All commands that are default in the bot start with a +, rather than a !, so that the commands don't clash with any other bots in the channel, but custom commands can begin with whatever you like.
Default commands are are +dogs, +about, adding custom emotes/commands, and it will post a dog emote if 3 lines of chat with the dog in them is posted in the current chat you are in.
You can disable the dog emote replies and the +dogs commands by typing '+disable dogs' in your chat.

Adding/Remove Modules

Modules are: speedrun, youtube, kadgar, dogs, and dogfacts.

+add 'module'

When you type this command into chat with a module name after it, it allows the use of the commands in those modules to be used in your channel.

+disable 'module'

When you type this command into chat with a module name after it, it disallows the use of the commands in those modules to be used in your channel.

Speedrun Commands

+wr gamecurrent game categoryvaries

This command gets the world record of the given game and category from speedrun.com api and posts it in the chat the command was used in.

+wr videocurrent game categoryvaries

This command gets the video for the world record of the given game and category from speedrun.com api and posts it in the chat the command was used in.

+pb runnercurrent channel gamecurrent game categoryvaries

This command gets the personal record of the given runner in the given game and category and posts the user's record in the chat the command was used in.

+splits runnercurrent channel gamecurrent game categoryvaries

This command gets the runner's splits for the given game and category from speedrun.com api and posts a link to the splits.io page for them.

+race multistreamkadgar usercurrent channel

This command gets a list of users in the race, if the given user is in a race, checks if the users are currently live on twitch, then posts a multitwitch link in chat for the live races. All options are srl, kadgar, speedruntv, and multitwitch.

Custom Stuff

+new command 'command' 'response'

This command lets you add custom commands to your channel. The command in quotes is what you would name your command, the response in quotes is what your response would be to the command. The response is the only thing that can have spaces in it.

+new emote 'emote'

This command lets you add custom emotes for the bot to respond to, similar to the dog emotes it replies after the emote is posted 3 times. The emote in quotes is the emote you want the bot to reply to.

YouTube

If a YouTube link is posted in the current channel you are in, the bot can recognize the link and post the Youtube video title and the channel who posted it.

Dog Facts

+dogfacts

When this command is used in a chat, a random fact about dogs, surrounded in dog emotes, is posted in the chat the command was used in.

Kadgar

+kadgar 'channel' 'channel'

This commands makes a kadgar link of the channels you type after +kadgar, it can be used for infinitely many channels, just seperate by spaces.
